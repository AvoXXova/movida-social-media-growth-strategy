# Movida Social Media Growth Strategy

Developed as part of an IIT Bombay Techfest Project.

## Project Overview

This project presents a 4-week social media growth and influencer marketing strategy for Movida, a premium lifestyle footwear brand targeting Gen Z consumers and young professionals.

## Key Deliverables

- Social Media Campaign Strategy
- Content Calendar Framework
- Influencer Marketing Plan
- UGC Campaign Design
- Budget & ROI Analysis
- Competitor Benchmarking
- Risk Management Strategy
